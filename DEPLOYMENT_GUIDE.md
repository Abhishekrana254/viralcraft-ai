# 🚀 INSTANT DEPLOYMENT GUIDE - Go Live in 5 Minutes!

## ⚡ **VERCEL DEPLOYMENT (FASTEST)**

### **Step 1: Push to GitHub (2 minutes)**
```bash
# Create GitHub repo first at github.com
git remote add origin https://github.com/YOUR_USERNAME/viralcraft-ai.git
git branch -M main  
git push -u origin main
```

### **Step 2: Deploy to Vercel (2 minutes)**
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project" 
3. Import your GitHub repo
4. **IMPORTANT**: Add Environment Variables:
   ```
   VITE_GEMINI_API_KEY=your_gemini_key
   VITE_HF_API_KEY=your_huggingface_key
   ```
5. Click "Deploy"
6. **DONE!** Live URL ready! 🔥

### **Step 3: Test Live App (1 minute)**
- Open your Vercel URL
- Test AI content generation
- **BOOM!** - Professional SaaS live!

## 🆓 **NETLIFY DEPLOYMENT (ALTERNATIVE)**

### **Drag & Drop Method:**
1. Run `npm run build`
2. Go to [netlify.com/drop](https://netlify.com/drop)
3. Drag `dist` folder
4. Add environment variables in Netlify dashboard
5. **Live!**
